## 为什么选择这些 Skills？

影视行业蓬勃发展，内容形式日益多元化——从竖屏短剧到长篇剧集，从网文IP到电影剧本，创作效率和质量成为核心竞争力。这 31 个精心设计的 Skills 专为影视创作者、策划师、内容评估师打造，将 AI 能力深度融入创作工作流，让每个环节都如虎添翼。

### 核心价值

- **提升效率 10 倍**：从小说筛选到剧本创作，一键完成
- **保证创作质量**：专业的评估体系，不放过任何一个优质 IP
- **降低创作门槛**：即使是新手，也能创作出专业级剧本
- **全程可控**：每个环节都可单独调用，精准掌控创作方向
- **多场景覆盖**：支持短剧、中长剧、长剧、电影等多种内容类型

## Skills 详细分类

### 一、小说初筛类 (4个)

> 每天面对海量小说投稿？这些 Skills 帮你快速锁定优质内容

| Skill | 功能 | 核心能力 | 适用类型 |
|-------|------|----------|----------|
| **novel-summarizer**<br/>故事大纲生成专家 | 阅读理解小说文本，总结成流畅的故事大纲 | - 500-800字精准总结<br/>- 提炼人物关系<br/>- 保持叙事完整性 | 所有类型 |
| **novel-evaluator**<br/>资深故事评估专家 | 对故事文本进行严格细致的判断与评分 | - 多维度评分体系<br/>- 改编潜力评估<br/>- 专业评审报告 | 所有类型 |
| **score-analyzer**<br/>评分分析智能体 | 分析多轮评估结果中的评分数据并计算评级等级 | - 智能数据聚合<br/>- 评级等级计算<br/>- 评估结果可视化 | 所有类型 |
| **novel-truncator**<br/>文本截断工具 | 智能截断文本，保持内容的完整性 | - 语义完整性保持<br/>- 智能断点识别<br/>- 长文本友好 | 所有类型 |

**使用场景**：
- 网文平台每日筛选 100+ 部作品
- 影视公司 IP 库快速评估
- 个人创作者寻找优质改编素材

### 二、故事分析类 (8个)

> 深度解构故事，挖掘每一个创作可能性

| Skill | 功能 | 核心能力 | 适用类型 |
|-------|------|----------|----------|

| **drama-analyzer**<br/>剧本分析专家 | 分析故事文本，提炼情节点并分析戏剧功能 | - 精准提炼情节点<br/>- 戏剧功能分类<br/>- 节奏把控分析 | 中长剧、长剧 |
| **plot-keypoints**<br/>大情节点分析专家 | 梳理故事主要脉络，提炼主要情节点 | - 三幕式结构识别<br/>- 核心情节点提炼<br/>- 故事骨架构建 | 所有类型 |
| **detailed-plot-analyzer**<br/>详细情节点分析专家 | 对大情节点进行深入分析和扩展 | - 情节点细化拆解<br/>- 戏剧张力分析<br/>- 情感曲线绘制 | 中长剧、长剧 |

## 实战案例：完整工作流演示

### 场景一：从小说到竖屏短剧剧本

```
┌──────────────────────────────────────────────────────────────┐
│  输入：一部10万字的都市言情小说                               │
│  目标：改编为竖屏短剧（80-100集，每集1-2分钟）                │
├──────────────────────────────────────────────────────────────┤
│                                                               │
│  Step 1: novel-summarizer                                    │
│  ─────────────────────                                       │
│  → 生成 500-800 字故事大纲                                   │
│                                                               │
│  Step 2: drama-evaluator                                     │
│  ─────────────────────                                       │
│  → 评估短剧改编潜力（爽点、人物、情节）                      │
│                                                               │
│  Step 3: drama-planner                                       │
│  ─────────────────────                                       │
│  → 短剧专业策划（情绪曲线、黄金三秒、付费卡点）              │
│                                                               │
│  Step 4: drama-creator                                       │
│  ─────────────────────                                       │
│  → 完整短剧剧本生成（情绪弹簧理论应用）                      │
│                                                               │
│  Step 5: script-evaluator                                    │
│  ─────────────────────                                       │
│  → 剧本质量评估与优化建议                                    │
│                                                               │
│  ──────────────────────────────────────────────────────────  │
│  输出：80-100集竖屏短剧剧本 + 评估报告                       │
│                                                               │
└──────────────────────────────────────────────────────────────┘
```

## 不同内容类型的推荐工作流

### 竖屏短剧推荐流程

```
小说初筛 → 改编评估 → 短剧策划 → 剧本创作 → 质量评估
novel-evaluator → drama-evaluator → drama-planner → drama-creator → script-evaluator
```

### 中长剧推荐流程

```
故事分析 → 情节规划 → 人物设计 → 大纲评估
story-five-elements → plot-workflow → character-profile → story-outline-evaluator
```

### 长剧/电视剧推荐流程

```
拉片学习 → 故事分析 → 详细情节 → 人物关系
series-analyzer → story-five-elements → detailed-plot-analyzer → character-relationships
```

### 电影剧本推荐流程

```
剧本评估 → 故事梗概 → 人物小传 → 优化建议
script-evaluator → story-summarizer → character-profile → script-evaluator
```
