## Why Choose These Skills?

The film and TV industry is booming, with increasingly diverse content formats—from vertical short dramas to long-form series, from web novel IPs to film scripts. Creative efficiency and quality have become core competitive advantages. These 31 carefully designed Skills are crafted for film and TV creators, planners, and content evaluators, deeply integrating AI capabilities into the creative workflow to supercharge every step.

### Core Value

- **10x Efficiency Boost**: One-click completion from novel screening to script creation
- **Guaranteed Creative Quality**: Professional evaluation systems that never miss a quality IP
- **Lower Creation Barrier**: Even beginners can create professional-grade scripts
- **Full Process Control**: Each step can be invoked independently for precise creative direction
- **Multi-Scenario Coverage**: Supports short dramas, medium-length dramas, long dramas, films, and more

## Recommended Workflows for Different Content Types

### Vertical Short Drama Recommended Process

```
Novel Screening → Adaptation Evaluation → Short Drama Planning → Script Creation → Quality Evaluation
novel-evaluator → drama-evaluator → drama-planner → drama-creator → script-evaluator
```

### Medium-Length Drama Recommended Process

```
Story Analysis → Plot Planning → Character Design → Outline Evaluation
story-five-elements → plot-workflow → character-profile → story-outline-evaluator
```

### Long Drama/TV Series Recommended Process

```
Scene-by-Scene Learning → Story Analysis → Detailed Plot → Character Relationships
series-analyzer → story-five-elements → detailed-plot-analyzer → character-relationships
```

### Film Script Recommended Process

```
Script Evaluation → Story Synopsis → Character Biography → Optimization Suggestions
script-evaluator → story-summarizer → character-profile → script-evaluator
```
