# Vertical Screen Drama Market Benchmark Analysis

## Market Overview

The vertical screen drama market is experiencing explosive growth, with market size reaching 300 billion yuan in 2023 and user scale exceeding 500 million. Major platforms include Douyin, Kuaishou, WeChat Video Account, etc.

## Hit Case Analysis

### Case 1: "Sweet Romance Counterattack"

**Basic Information**
- Genre: Sweet Romance + Slap + Counterattack
- Episodes: 80 episodes
- Views: 520 million
- Rating: 8.6

**Core Satisfaction Points**
- Hidden identity rich second-generation CEO
- Snobbish ex-boyfriend gets slapped
- Extreme satisfaction from identity reveal

**Success Elements**
1. **Extreme Identity Contrast**: Delivery rider → Group Chairman
2. **Classic Slap Scene**: Ex-boyfriend humiliation → Identity reveal → Instant slap
3. **Clear Emotional Line**: Sweet romance line throughout
4. **Tight Rhythm**: Small climax in every episode

**Learnable Experience**
- Identity contrast is a classic satisfaction point, proper execution can create a hit
- Slap scenes need extreme contrast and strong performance
- Sweet romance elements can increase female viewer stickiness

### Case 2: "Reborn Revenge Queen"

**Basic Information**
- Genre: Rebirth + Revenge + Slap
- Episodes: 100 episodes
- Views: 870 million
- Rating: 8.9

**Core Satisfaction Points**
- Rebirth after victimization, revenge with memory
- Collect evidence, carefully plot
- Finally reveal truth, villains punished

**Success Elements**
1. **Attractive Rebirth Setup**: Victimized in past life, revenge in this life
2. **Tight Revenge Rhythm**: Progressive from plotting to revenge
3. **Strong Satisfaction Release**: Extreme revenge scene design
4. **Auxiliary Emotional Line**: Romantic line with male lead adds warmth

**Learnable Experience**
- Rebirth genre has built-in audience base
- Revenge line needs rhythm, can't be continuously suppressive
- Emotional line can neutralize the coldness of revenge

### Case 3: "Palace Intrigue Rise to Power"

**Basic Information**
- Genre: Costume + Palace Intrigue + Counterattack
- Episodes: 120 episodes
- Views: 630 million
- Rating: 8.4

**Core Satisfaction Points**
- Counterattack from selected maiden to Empress
- Wits battles with Empress, step-by-step plotting
- Successfully rise to power

**Success Elements**
1. **Complete Growth Line**: From innocent to masterful schemer
2. **Brilliant Wits Battles**: Clever design of palace struggles
3. **Complex Character Relationships**: Multiple relationships interwoven
4. **Magnificent Scenes**: Strong visual impact of palace scenes

**Learnable Experience**
- Growth line is core of costume palace intrigue
- Wits battles need both rationality and dramatic quality
- More complex character relationships, stronger dramatic tension

## User Profile Analysis

### Core Audience
- **Age**: 18-35 years old account for 78%
- **Gender**: Females account for 65%
- **Region**: Second and third-tier cities account for 60%
- **Spending Power**: Monthly payment willingness 30-50 yuan

### Viewing Habits
- **Viewing Time**: 8-11 PM accounts for 55%
- **Viewing Scenes**: Commuting, Resting, Before Sleep
- **Viewing Duration**: 15-30 minutes each time
- **Payment Willingness**: Willing to pay for satisfactory content

## Hit Formula Analysis

### Sweet Romance Slap Type
**Core Formula**: Hidden Identity + Snobbish Antagonist + Extreme Slap + Sweet Romance Support

**Essential Elements**
1. Protagonist has hidden identity (tycoon, big shot, executive)
2. Antagonist extremely arrogant, snobbish
3. Dramatized identity reveal scene
4. Sweet romance line throughout

### Rebirth Revenge Type
**Core Formula**: Past Life Victimization + Rebirth Awakening + Careful Plotting + Successful Revenge

**Essential Elements**
1. Clear past life victimization plot
2. Immediately start plotting after rebirth
3. Tense and exciting evidence collection process
4. Revenge climax satisfaction release

### Costume Palace Intrigue Type

**Essential Elements**

## Benchmark Evaluation Points

### Evaluation Standards
1. **Whether Genre is Hot**: Benchmark against market hot genres
2. **Whether Satisfaction Points are Clear**: Whether core satisfaction points are clear
3. **Whether Rhythm is Tight**: Whether it fits vertical screen drama rhythm
4. **Whether Characters are Extreme**: Whether character settings have memorable points
5. **Whether Plot is Brilliant**: Whether plot design has tension

### Optimization Directions
1. **Genre Selection**: Prioritize hot genres
2. **Satisfaction Point Strengthening**: Enhance core satisfaction point performance
3. **Rhythm Optimization**: Speed up rhythm, increase suspense
4. **Character Extremization**: Make character settings more extreme
5. **Plot Innovation**: Innovate within classic framework
