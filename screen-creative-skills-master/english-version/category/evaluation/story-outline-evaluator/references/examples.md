# Story Outline Evaluation Expert - Detailed Examples

## Example 1: Urban Romance Outline Evaluation

### Input

```
Please evaluate the following story outline:

[Outline content]
Lin Qian and Lu Yuan develop from competitors to lovers at an advertising company. The script portrays their journey from confrontation to mutual appreciation at work. The appearance of an ex-girlfriend creates tests, but they ultimately overcome obstacles and get together.
```

### Output

```
[Story Outline Evaluation Report]

I. Outline Structure Analysis
- Structural Completeness: Complete, includes beginning, development, climax, ending
- Plot Clarity: Clear, plot development context is clear
- Logical Coherence: Coherent logic, reasonable causal relationships

II. Content Evaluation
- Story Creativity: Urban romance genre, average creativity
- Character Settings: Protagonist settings clear, supporting characters thin
- Plot Design: Reasonable plot development, but lacks surprises
- Emotional Thread: Clear emotional thread, but insufficient tension

III. Adaptation Potential Evaluation
- Film and TV Adaptation Value: Medium, suitable for web series or short films
- Commercialization Potential: Medium, accurate audience positioning but fierce competition
- Development Recommendations: Suggest adding dramatic conflicts and surprise elements

[Overall Score]: 7.5/10

[Improvement Suggestions]
1. Add supporting character layers, enrich character relationships
2. Enhance dramatic quality and conflict intensity of plot
3. Add more twists in emotional thread
```
