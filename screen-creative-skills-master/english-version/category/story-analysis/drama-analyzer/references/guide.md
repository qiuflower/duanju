# Script Breakdown Analysis Guide

## I. Core Definitions

### 1. Plot Point
A plot point is a key plot or event in a story.
- **Function**: Has significant impact on plot development and emotional relationships between characters.
- **Characteristics**: Usually a turning point in the story, can trigger plot changes, conflict escalation, or character growth.

### 2. Dramatic Function
Dramatic function refers to the role and impact of plot in the story.
- **Specific Functions**: Drive story development, shape characters, reveal themes, create conflict or tension, stimulate audience interest.
- **Value**: Enhance story quality and effectiveness through these methods, making it more attractive.

## II. Analysis Dimensions

### 1. Deep Reading
Conduct deep reading and understanding of story text, grasp overall context.

### 2. Extract Plot Points
- **Identify**: Find key events that drive story development, ignore trivial transitional scenes.
- **Summarize**: Use refined language (no more than 100 words) to summarize each plot point's content.
- **Quantity**: Extract at least 5 major plot points.

### 3. Analyze Dramatic Function
- **Qualify**: Analyze what role the plot point serves (e.g., establish suspense, inciting incident, crisis, climax, resolution).
- **Connect**: Analyze how the plot point connects preceding and succeeding, how it affects character relationships.

## III. Workflow

1. **Text Reading**: Read full text from senior screenwriter perspective.
2. **Plot Point Capture**: Mark all turning points and key events.
3. **Function Breakdown**: Conduct dramatic function analysis on each marked point.
4. **Structured Output**: Output report in [Plot Point] + [Dramatic Function] format.

## IV. Notes

- **Avoid Hallucination**: Strictly based on text content, do not fabricate plots.
- **Objective and Accurate**: Faithful to original work's logic and settings.
- **Format Specification**: Do not use Arabic numerals for numbering, maintain clear paragraph structure.
