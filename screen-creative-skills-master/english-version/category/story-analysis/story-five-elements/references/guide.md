# Story Five Elements Analysis Guide

## Five Elements Overview

Story five elements are the basic framework for story analysis and creation, including:
1. **Genre Type and Creative Extraction**
2. **Story Synopsis**
3. **Character Biographies**
4. **Character Relationships**
5. **Major Plot Points**

These five elements are interrelated, together forming a complete story system.

## I. Genre Type and Creative Extraction

### Genre Type Classification

**By Era**
- Ancient: Costume, fantasy, wuxia, xianxia
- Modern: Urban, workplace, youth, campus
- Future: Sci-fi, apocalyptic, cyberpunk

**By Style**
- Comedy: Light and humorous
- Tragedy: Heavy and moving
- Drama: Serious and earnest
- Suspense: Tense and thrilling

**By Content**
- Romance: Sweet romance, abusive love, secret crush
- Counterattack: Workplace counterattack, class crossing
- Revenge: Rebirth revenge, present revenge
- Growth: Character growth, self-realization

### Creative Extraction Points

**Core Concept**
- Extract story's most core concept or setting
- Summarize story's uniqueness in one sentence
- Emphasize story's innovation points and memorable points

**Creative Elements**
- Supernatural settings: Rebirth, transmigration, system
- Special identities: Hidden big shot, mysterious background
- Special abilities: Superpowers, talents, skills
- Special relationships: Fate, blood ties, past and present lives

**Style Characteristics**
- Narrative style: Light, heavy, humorous, serious
- Visual style: Magnificent, minimalist, aesthetic, realistic
- Emotional style: Warm, cold, complex, pure

## II. Story Synopsis

### Synopsis Writing Points

**Word Count Control**
- Complete synopsis: 300-500 words
- Short synopsis: 150-200 words
- One-sentence synopsis: 30-50 words

**Structure Requirements**
- Beginning: Introduce protagonist and background
- Middle: Describe main conflicts and development
- End: Explain final result

**Content Elements**
- Who is the protagonist?
- What does the protagonist want?
- What's obstructing the protagonist?
- How does protagonist overcome obstacles?
- What's the final result?

### Synopsis Writing Techniques

**Technique 1: Focus on Mainline**
- Only tell mainline story
- Omit支lines and supporting characters
- Highlight core conflicts

**Technique 2: Concise and Powerful**
- Use concise language to summarize
- Avoid detailed descriptions
- Highlight key turning points

**Technique 3: Emotionally Rich**
- Convey story emotions
- Trigger reader resonance
- Showcase story charm

## III. Character Biographies

### Biography Structure

**Basic Information**
- Name, age, identity
- Appearance features, personality traits

**Background Story**
- Family background, growth experience
- Important experiences, turning point events

**Character Traits**
- Personality traits, behavior patterns
- Language style, thinking patterns

**Character Arc**
- State at story beginning
- Growth and changes during story
- State at story ending

### Biography Writing Techniques

**Technique 1: Three-Dimensional**
- Multi-dimensionally showcase characters
- Highlight character complexity
- Avoid caricature

**Technique 2: Motivation**
- Clarify character motivations
- Explain behavior reasons
- Establish inner logic

**Technique 3: Contrast**
- External vs. internal contrast
- Past vs. present contrast
- With other characters contrast

## IV. Character Relationships

### Relationship Types

**Emotional Relationships**
- Family: Parents, siblings
- Romance: Lovers, spouses, exes
- Friendship: Friends, classmates, comrades-in-arms

**Interest Relationships**
- Cooperation: Partners, allies
- Opposition: Competitors, enemies
- Dependence: Superior-subordinate, master-servant

**Power Relationships**
- Domination: Controller, controlled
- Influence: Influencer, influenced
- Status: High class, low class

### Relationship Characteristics

**Relationship Strength**
- Strong binding: Blood, true love, life-and-death friendship
- Medium binding: Colleagues, classmates, acquaintances
- Weak binding: Strangers, casual acquaintances

**Relationship Tension**
- Conflict Type: Life-and-death, incompatible
- Competitive Type: Mutual competition, trade-off
- Complementary Type: Mutual complementation, win-win cooperation
- Dependent Type: Mutual dependence, symbiotic coexistence

**Relationship Development**
- Stable Type: Relationship remains stable
- Developmental Type: Relationship gradually deepens or deteriorates
- Transformative Type: Relationship undergoes qualitative change

### Relationship Network Analysis

**Core Relationships**
- Protagonist's relationships with core characters
- Relationships driving mainline development
- Relationships affecting story direction

**Secondary Relationships**
- Auxiliary character relationships
- Relationships enriching story layers
- Relationships providing background information

**Potential Relationships**
- Not yet revealed relationships
- Possible developmental relationships
- Potential conflicts and cooperation

## V. Major Plot Points

### Plot Point Classification

**By Function**
- Establishment plot points: Settings and background
- Development plot points: Conflicts and challenges
- Turning plot points: Accidents and changes
- Climax plot points: Final confrontation
- Resolution plot points: Conflict resolution

**By Stage**
- Early plot points: Story beginning stage
- Middle plot points: Story development stage
- Late plot points: Story ending stage

### Plot Point Organization

**Organization Methods**
- Organize by chronological order
- Organize by cause-effect relationships
- Organize by theme deepening

**Organization Points**
- Identify key turning points
- Identify climax points
- Identify resolution points
- Ensure logical coherence

## Five Elements Integration Analysis

### Integration Points

**Consistency Check**
- Whether five elements support each other
- Whether contradictions and conflicts exist
- Whether unified whole is formed

**Completeness Check**
- Whether five elements are complete
- Whether information is complete
- Whether omissions exist

**Accuracy Check**
- Whether based on original text analysis
- Whether accurately understand story
- Whether subjective judgments exist

### Integration Methods

**Method 1: Theme Leadership**
Ensure five elements unfold around theme with theme as leadership.

**Method 2: Character Driven**
Ensure five elements serve character growth with characters as drivers.

**Method 3: Plot Connection**
Ensure five elements connect through plot with plot as connection.

## Analysis Process

### Step 1: Read Full Text
- Understand story overall
- Grasp core theme
- Identify key information

### Step 2: Classification and Extraction
- Extract genre type information
- Extract character information
- Extract plot information
- Extract relationship information

### Step 3: Analysis and Organization
- Analyze genre characteristics
- Organize character biographies
- Organize character relationships
- Extract major plot points

### Step 4: Integrated Output
- Integrate five elements analysis
- Generate structured report
- Ensure logical coherence

## Common Problems

### Problem 1: Incomplete Information
**Solution**: Carefully read text, extract all related information

### Problem 2: Surface Analysis
**Solution**: Deep analysis, mine deep meanings

### Problem 3: Confused Logic
**Solution**: Organize according to logical order, ensure coherence

### Problem 4: Subjective Judgment
**Solution**: Analyze based on original text, do not create

## Best Practices

### Practice 1: Multiple Reading
Read original text at least twice, first time to understand overall, second time to extract information.

### Practice 2: Note Taking
Take notes while reading, mark key information.

### Practice 3: Cross Validation
Cross validate between different elements, ensure consistency.

### Practice 4: Repeated Revision
After analysis completion, repeatedly revise and polish, improve quality.
