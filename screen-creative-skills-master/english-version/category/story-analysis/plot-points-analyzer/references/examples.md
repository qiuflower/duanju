# Plot Point Analysis Expert - Detailed Examples

## Example 1: Analyze Betrayal Revelation Plot

### Input

```
Analyze the following plot point:
"Protagonist discovers trusted best friend is actually the mastermind behind everything"
```

### Output

```
[Plot Point]: Betrayal Truth Revealed

1. Plot Point Type
   - Type: Turning Point
   - Function: Key reversal in story middle, breaks early established character relationship pattern

2. Plot Point Importance
   - Importance Level: Core
   - Scope: Influences entire story后半段走向 and protagonist's all action motivations

3. Plot Development Process
   - Prerequisites: Protagonist and best friend established deep trust relationship
   - Development: Through series of clue discoveries, protagonist gradually generates suspicion, finally confirms truth in confrontation
   - Result Orientation: Protagonist falls into trust crisis, emotions severely damaged

4. Turning Point Analysis
   - Turning Nature: Negative
   - Turning Mechanism: Achieved through information gap and perspective shift
   - Emotional Impact: Produces strong shock and sympathy

5. Plot's Impact on Story
   - On Character Development: Forces protagonist to shift from relying on others to independent growth
   - On Main Plot: Shifts confrontation target from external to internal
```
