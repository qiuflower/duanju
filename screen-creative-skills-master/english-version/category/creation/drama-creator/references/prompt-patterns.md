# Vertical Screen Drama Creation Common Prompt Patterns

This file provides common prompt patterns for different creation tasks, helping users interact more effectively with the skill.

## Task Mode Prompt Patterns

### 1. Macro Construction Mode

**Purpose**: Design overall story outline, plan long-form plot structure, layout core paywall points

**Prompt Template**:
```
Based on the following idea, design a [number of episodes]-episode vertical screen drama outline:
[Idea description/Core concept]

Requirements:
- Clarify core satisfaction points and main emotions
- Plan emotional curve (press-release rhythm)
- Design key paywall point locations
- Create character profiles
```

**Example**:
```
Based on the following idea, design a 10-episode vertical screen drama outline:
A hidden identity God of War CEO pretends to be poor to experience life, gets dumped by girlfriend for being poor, later identity revealed, ex-girlfriend deeply regrets.

Requirements:
- Clarify core satisfaction points and main emotions
- Plan emotional curve (press-release rhythm)
- Design key paywall point locations
- Create character profiles
```

---

### 2. Script Creation Mode

**Purpose**: Write complete single or multi-episode scripts

**Prompt Template**:
```
Create episode [N] script, plot points:
[Specific plot description]

Requirements:
- This episode is a [press spring/release spring] episode
- Follow information-first, action visualization principles
- Set opening hook and ending suspense
- Word count within 800
```

**Example**:
```
Create episode 5 script, plot points:
Male lead is humiliated by ex-girlfriend and her new boyfriend at restaurant, restaurant manager recognizes male lead's identity, publicly reveals male lead is group chairman.

Requirements:
- This episode is a release spring episode
- Follow information-first, action visualization principles
- Set opening hook and ending suspense
- Word count within 800
```

---

### 3. Precision Optimization Mode

**Purpose**: Plot modification, rhythm adjustment, dialogue polishing for existing scripts

**Prompt Template**:
```
Optimize the following script:
[Original script content]

Optimization directions:
- [Specific problem 1]
- [Specific problem 2]

Requirements:
- Explain modification reasons
- Compare before and after effects
```

**Example**:
```
Optimize the following script:
1. Cafe Day Interior
Xiao Li is drinking coffee, colleague Xiao Zhang walks over.

Xiao Zhang: I heard you're resigning?

Xiao Li: Yes, I want to change environments.

Xiao Zhang: Where are you going?

Xiao Li: Not sure yet, maybe start a business.

Xiao Zhang: Good luck, I support you.

Xiao Li: Thanks.

Optimization directions:
- Dialogue flat, lacking conflict
- No suspense, no emotional peak
- Insufficient information, no attraction points

Requirements:
- Explain modification reasons
- Compare before and after effects
```

---

### 4. Ideation Mode

**Purpose**: Conceive "slap" scenes, design suspense hooks, brainstorm character profiles

**Prompt Template**:
```
Help me conceive [quantity] [type] creative ideas:

Theme: [Story theme]
Requirements:
- [Requirement 1]
- [Requirement 2]
```

**Example**:
```
Help me conceive 5 slap scene creative ideas:

Theme: Hidden big shot identity revealed
Requirements:
- Each scene has different reveal method
- Shock level escalates
- Suitable for vertical screen drama format
```

## Prompt Patterns by Creation Phase

### Pre-Planning Phase

#### Emotional Value Analysis Prompt
```
Analyze the core emotional value of the following story idea:

[Idea description]

Please analyze:
1. What is the core emotion (satisfaction/sweet/angst/suspense/resonance/warmth)?
2. What are the emotional trigger points?
3. How to design the emotional curve?
```

#### Character Design Prompt
```
Design a [type] protagonist persona:

Story background: [Background description]

Requirements:
- Design 3 core tags
- Tags must be extreme
- Clarify persona container function (what desires/emotions/fantasies it carries)
- Design contrast (surface vs. true)
```

---

### Mid-Creation Phase

#### Opening Hook Design Prompt
```
Design a [type] opening hook (first 3 seconds):

Story background: [Background description]

Requirements:
- First second: Visual impact
- Second second: Dialogue impact
- Third second: Throw out suspense
```

#### Conflict Scene Design Prompt
```
Design a conflict scene:

Scene type: [misunderstanding/confrontation/humiliation/crisis]
Participants: [Character list]
Emotional goal: [Press spring/Release spring]

Requirements:
- Action visualization
- Information first
- Strong emotions
```

#### Identity Reveal Scene Design Prompt
```
Design an identity reveal scene:

Hidden identity: [Specific identity]
Reveal scene: [Location, occasion]
People present: [Character list]

Requirements:
- Layered reveal (foreshadowing-hinting-reveal)
- Multi-perspective reactions (protagonist/supporting/antagonist)
- Maximize shock effect
```

---

### Post-Optimization Phase

#### Rhythm Check Prompt
```
Check the following script for rhythm issues:

[Script content]

Please analyze:
1. Are there dragging plot points?
2. Are hooks attractive?
3. Is emotional function clear?
4. How to optimize rhythm?
```

#### Dialogue Optimization Prompt
```
Optimize dialogue for the following scene:

[Original script segment]

Optimization goals:
- Information first
- Delete fluff
- Match character personality
- Strengthen conflict or emotion
```

#### Suspense Design Prompt
```
Design ending suspense for the following scene:

[Scene description]

Requirements:
- Suspense type: [identity/truth/crisis/reversal]
- Must make audience want to watch next episode
- Related to main plot
```

## Prompt Patterns by Drama Type

### Urban Counterattack Type
```
Create an urban counterattack type script:

Core setup: [Protagonist identity + counterattack reason]

Requirements:
- First 3 episodes continuous press spring (accumulate negative emotions)
- Episodes 4-5 start release spring (identity reveal)
- Design at least 3 slap scenes
- Antagonists must be arrogant enough to make audience want to hit them
```

### Sweet Romance Type
```
Create a sweet romance type script:

Core setup: [Male and female lead relationship]

Requirements:
- Male lead's doting must be extreme
- Design misunderstanding-resolution cycles
- Sweet-angst-sweet emotional curve
- Small sweet moments in each episode
```

### Suspense Mystery Type
```
Create a suspense mystery type script:

Core mystery: [Mystery description]

Requirements:
- New clues in each episode
- Truth hidden until final episode
- Set fake clues to mislead
- Danger throughout
```

## Quick Iterative Optimization Prompts

### Iterative Optimization Mode 1: Direct Problem Identification
```
What problems does this script segment have? Please point out directly:

[Script content]

Please analyze from the following dimensions:
- Emotional function
- Rhythm issues
- Dialogue issues
- Format standards
```

### Iterative Optimization Mode 2: Targeted Modification
```
Please modify the following script targeting [specific problem]:

[Script content]

Modification requirements:
- Keep other parts unchanged
- Explain modification reasons
```

### Iterative Optimization Mode 3: Comparison Version
```
Please provide two versions of the script:

Original scene: [Scene description]

Version 1: Conservative modification (minor optimization)
Version 2: Radical modification (complete reconstruction)

Please compare and explain pros and cons of both versions.
```

## Prompt Writing Techniques

### Technique 1: Clarify Emotional Function
In the prompt, clearly indicate whether this scene is "press spring" or "release spring," which helps the skill better grasp emotional direction.

### Technique 2: Provide Specific Plot
The more specific the plot description, the better - avoid vague expressions.
- ❌ "Write a slap scene"
- ✅ "Write a restaurant slap scene where ex-girlfriend humiliates male lead, manager reveals male lead's identity"

### Technique 3: Set Constraint Conditions
Clear constraint conditions yield more precise output:
- Word limit: "Within 800 words"
- Duration limit: "Within 2 minutes"
- Scene limit: "No more than 3 scenes"

### Technique 4: Indicate Creation Style
Tell the skill the expected style:
- "Dialogue should be sharp"
- "Rhythm should be tight"
- "Reversals should be unexpected"

### Technique 5: Utilize Iteration
If first output isn't satisfactory, continue iterating:
- "This version is too mild, please strengthen conflict"
- "Dialogue still has fluff, please further streamline"
- "Suspense not strong enough, please design stronger ending hook"
