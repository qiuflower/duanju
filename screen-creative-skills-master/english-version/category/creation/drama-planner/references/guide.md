# Vertical Screen Drama Planning Guide

## Core Planning Philosophy

### 1. Emotional Value First Principle

**Core Idea**: The essence of vertical screen dramas is emotional products. Success lies in accurately and efficiently identifying and triggering target audience's core emotional needs.

#### Identifying Emotional Value Points

**Six Core Emotions**:
1. **Satisfaction**: Slap, counterattack, revenge, retaliation
2. **Sweetness**: Doting, protection, giving, sacrifice
3. **Angst**: Misunderstanding, separation, betrayal, harm
4. **Suspense**: Questions, anticipation, curiosity, exploration
5. **Resonance**: Reality, immersion, understanding, identification
6. **Warmth**: Healing, growth, family, friendship

**Emotional Value Assessment Dimensions**:
- **Universality**: Is this emotion widely understood and recognized by the masses
- **Intensity**: Can this emotion be strongly triggered
- **Persistence**: Can this emotion last for a considerable time
- **Replicability**: Can this emotional pattern be used repeatedly

#### Emotional Trigger Mechanisms

**Three-Level Trigger System**:
1. **Direct Trigger**: Directly trigger emotions through intuitive plots (e.g., slap)
2. **Indirect Trigger**: Trigger emotions through hints and foreshadowing (e.g., identity hints)
3. **Chain Trigger**: Trigger emotions through multiple stacked plots (e.g., continuous reversals)

**Trigger Techniques**:
- **Contrast Trigger**: Before-and-after comparison, strengthen emotion (e.g., from being stepped on to stepping on others)
- **Accumulation Trigger**: Small emotions accumulate into big emotions (e.g., explosion after continuous humiliation)
- **Surprise Trigger**: Unexpected, strengthen impact (e.g., hidden identity reveal)

### 2. Golden Three Seconds Hook Rule

**Core Idea**: The first 3 seconds determine whether users continue watching, must quickly grab attention.

#### Three-Second Design Principles

**Visual Impact**:
- First frame is high-energy scene
- Avoid slow build-up or flat opening
- Use action, conflict, suspense to create visual impact

**Auditory Stimulation**:
- Use intense dialogue or sound effects
- Throw out heart-striking questions
- Create tense dialogue atmosphere

**Information Density**:
- Deliver core conflict within 3 seconds
- Quickly establish character relationships
- Clarify current plot state

#### Hook Type Library

**Conflict Type Hooks**:
- Two people confronting, about to fight
- One person accusing another
- Sudden quarrel or dispute

**Suspense Type Hooks**:
- Someone's identity unclear
- Something's truth unknown
- Something's use unknown

**Crisis Type Hooks**:
- Protagonist in danger
- Important character threatened
- Key item lost

**Reversal Type Hooks**:
- Situation beyond expectation
- Sudden identity reveal
- Sudden relationship change

#### Three-Second Design Templates

**Template 1: Conflict Opening**
```
[First second] Visual impact (e.g., a slap in the face)
[Second second] Intense dialogue ("You actually...")
[Third second] Throw out suspense (Why slapped?)
```

**Template 2: Suspense Opening**
```
[First second] Mysterious scene (e.g., mysterious person's back)
[Second second] Raise question ("Who is he?")
[Third second] Imply consequence ("He will change everything")
```

**Template 3: Crisis Opening**
```
[First second] Danger occurs (e.g., car accident, kidnapping)
[Second second] Character reaction (terror, struggle)
[Third second] Suspense ending (Can they escape?)
```

### 3. Expectation-Suppression-Explosion Three-Act Structure

**Core Idea**: A complete story should experience three phases: establishing expectation, accumulating suppression, and final explosion.

#### First Act: Establish Expectation

**Function**:
- Introduce protagonist and goal
- Establish story background
- Plant conflict seeds

**Duration**: About 15-20% of entire drama

**Key Elements**:
- **Protagonist Introduction**: Show protagonist's initial state
- **Goal Establishment**: Clarify what protagonist wants
- **Obstacle Foreshadowing**: Hint at upcoming difficulties

#### Second Act: Accumulate Suppression

**Function**:
- Continuously apply pressure
- Accumulate negative emotions
- Create anticipation of explosion

**Duration**: About 60-70% of entire drama

**Key Elements**:
- **Conflict Escalation**: From small conflicts to big conflicts
- **Emotional Accumulation**: From small suppression to big suppression
- **Reversal Setup**: Set small reversals within suppression

**Suppression Techniques**:
- **Addition Principle**: Continuously add new pressures to protagonist
- **Subtraction Principle**: Continuously deprive protagonist of support and resources
- **Delay Principle**: Continuously delay emotional release timing

#### Third Act: Explosive Release

**Function**:
- Release all accumulated emotions
- Resolve main conflicts
- Reach emotional climax

**Duration**: About 15-20% of entire drama

**Key Elements**:
- **Comprehensive Counterattack**: Protagonist counterattacks all antagonists
- **Truth Revealed**: Expose all concealed truths
- **Ultimate Reversal**: Set biggest reversal plot

### 4. Persona as Container Theory

**Core Idea**: Protagonist persona is the container carrying audience emotional projection, must be tagged and extreme.

#### Persona Tagging

**Tag Function**:
- Quick Recognition: Let audience quickly remember character
- Emotional Projection: Facilitate audience emotional immersion
- Viral Effect: Facilitate social media spread

**Tag Types**:
- **Identity Tags**: God of War, CEO, Doctor, Lawyer, etc.
- **Ability Tags**: Invincible, all-knowing, precognitive, mind-reading, etc.
- **Personality Tags**: Cold, gentle, manipulative, honest, etc.
- **Situation Tags**: Fallen, hidden, pretending poor, undercover, etc.

#### Persona Extremization

**Extremization Principles**:
- **Ability Extreme**: Reach extreme in some aspect
- **Personality Extreme**: Some personality trait extremely prominent
- **Contrast Extreme**: Surface vs. reality extremely different

**Extremization Examples**:
- **God of War Extreme**: From invincible to defeated to return invincible
- **Doting Extreme**: Dote to extreme, unconditional giving
- **Manipulative Extreme**: Surface gentle, actually ruthless methods

#### Persona Container Design

**Container Functions**:
1. **Carry Desires**: Carry audience's unfulfilled desires
2. **Carry Emotions**: Carry audience's suppressed emotions
3. **Carry Fantasies**: Carry audience's beautiful fantasies

**Design Points**:
- Container must be "big" enough (can hold various emotions)
- Container must be "deep" enough (can hold deep emotions)
- Container must be "stable" enough (consistent persona)

---

### 5. Commercial Paywall Logic

**Core Idea**: Set paywalls at plot climax points to maximize commercial value.

#### Paywall Setting Principles

**Paywall Position**:
- **Before Emotional Climax**: Paywall just before release
- **Before Suspense Reveal**: Paywall just before truth exposed
- **Before Reversal Occurs**: Paywall just before reversal happens

**Paywall Strength**:
- **Weak Paywall**: Just plot interruption, can continue watching
- **Medium Paywall**: Plot interrupted but can guess what follows
- **Strong Paywall**: Plot interrupted and cannot predict what follows

#### Paywall Design Techniques

**Natural Transition**:
- Don't make paywall seem abrupt
- Set at natural plot turning points
- Make payment an active audience choice

**Value Presentation**:
- Must have high-energy plot before paywall
- Make audience feel payment is worth it
- Provide exceed-expectation experience after payment

**Differentiated Design**:
- Different paywalls for different episodes
- Different paywalls within same episode
- Avoid audience fatigue

## Planning Proposal Development Process

### Step 1: Requirement Analysis

1. **Clarify Planning Goal**
   - Pre-planning or mid-adjustment
   - Overall proposal or single-item proposal
   - Target audience and expected effect

2. **Analyze Existing Materials**
   - Story idea or outline
   - Target audience positioning
   - Commercialization goals

### Step 2: Emotional Value Analysis

1. **Identify Core Emotional Value Points**
   - What's the main emotion (satisfaction/sweet/angst, etc.)
   - How to assess emotional intensity
   - How to ensure emotional persistence

2. **Design Emotional Trigger Mechanisms**
   - What are direct trigger points
   - What are indirect trigger points
   - How to design chain triggers

3. **Draw Emotional Curve**
   - X-axis: Time/Episode number
   - Y-axis: Emotional intensity
   - Mark key trigger points

### Step 3: Golden Three Seconds Hook Design

1. **Determine Hook Type**
   - Conflict type, suspense type, crisis type or reversal type
   - Opening method selection
   - Audio-visual effect design

2. **Design Three-Second Content**
   - First second: Visual impact
   - Second second: Dialogue impact
   - Third second: Suspense throw

3. **Optimize Hook Effect**
   - Test hook attraction
   - Adjust hook strength
   - Ensure hook relates to main plot

### Step 4: Three-Act Structure Planning

1. **First Act: Establish Expectation**
   - Introduce protagonist and goal
   - Plant conflict seeds
   - Foreshadow difficulties and obstacles

2. **Second Act: Accumulate Suppression**
   - Design conflict escalation path
   - Plan emotional accumulation curve
   - Set small reversals to adjust rhythm

3. **Third Act: Explosive Release**
   - Design comprehensive counterattack plot
   - Plan truth reveal method
   - Set ultimate reversal

### Step 5: Persona Container Design

1. **Protagonist Persona Design**
   - Core tag selection
   - Extremization degree setting
   - Container function clarification

2. **Supporting Character Persona Design**
   - Functional positioning (assist/obstruct/contrast)
   - Relationship setting with protagonist
   - Own emotional value points

3. **Antagonist Persona Design**
   - Opposition attribute setting
   - Suppression method planning
   - Failure mode design

### Step 6: Commercial Paywall Setup

1. **Determine Paywall Position**
   - Before plot climax
   - Before suspense reveal
   - Before reversal occurs

2. **Design Paywall Strength**
   - Determine based on plot importance
   - Consider user acceptance
   - Balance experience and revenue

3. **Optimize Paywall Experience**
   - Ensure natural transition
   - Provide value perception
   - Differentiated design

## Planning Proposal Assessment Standards

### Emotional Value Assessment

- **Is Core Emotion Clear**: Is there a clear main emotion
- **Is Emotional Intensity Sufficient**: Can it strongly trigger target emotions
- **Is Emotional Curve Reasonable**: Does it have ups and downs and climaxes

### Hook Design Assessment

- **Do Three Seconds Grab Attention**: Does opening have sufficient impact
- **Is Hook Type Appropriate**: Does hook type fit this drama type
- **Is Suspense Setup Effective**: Does it make people want to continue watching

### Structure Planning Assessment

- **Is Three-Act Structure Complete**: Is expectation-suppression-explosion complete
- **Is Rhythm Reasonable**: Is it tension-relax balanced, not dragging
- **Is Climax Powerful**: Is emotional release strong enough

### Persona Design Assessment

- **Are Tags Clear**: Is it easy to remember
- **Is Extremization in Place**: Has it reached extreme level
- **Is Container Function Clear**: Can it carry emotional projection

### Commercialization Assessment

- **Are Paywall Positions Appropriate**: Are they in best positions
- **Is Paywall Strength Reasonable**: Can it balance experience and revenue
- **Is Payment Willingness Strong**: Will audiences pay for paywalls

## Common Problems and Solutions

### Problem 1: Unclear Emotional Value

**Manifestation**: Audience doesn't know what emotions to expect

**Solution**:
- Clarify main emotion type
- Strengthen core emotional points
- Reduce interfering emotions

### Problem 2: Weak Hook

**Manifestation**: Flat opening, cannot attract continued viewing

**Solution**:
- Enhance visual impact
- Strengthen dialogue conflict
- Optimize suspense setup

### Problem 3: Chaotic Structure

**Manifestation**: Three-act structure unclear, emotional curve chaotic

**Solution**:
- Clarify three-act division
- Draw emotional curve
- Adjust plot rhythm

### Problem 4: Indistinct Persona

**Manifestation**: Character tags unclear, hard to remember

**Solution**:
- Strengthen core tags
- Extremize traits
- Clarify container function

### Problem 5: Abrupt Paywalls

**Manifestation**: Paywall points abrupt, affect experience

**Solution**:
- Optimize paywall positions
- Strengthen pre-paywall plot
- Natural transition handling
