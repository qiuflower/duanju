```

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Long Text Processing Flow
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Use: drama-workflow

AI: Detected long text, I use drama-workflow to process...

Automatic orchestration execution:

1. Text Preprocessing
   - text-truncator: Intelligently truncate to appropriate length
   - text-splitter: Split into processable text chunks

2. Parallel Analysis
   - Execute drama-analyzer in parallel for each text chunk
   - Analyze plot points and dramatic functions

3. Result Integration
   - result-integrator: Integrate all analysis results
   - Remove duplicate content
   - Classify by dramatic function
   - Sort by story order

4. Report Generation
   - Generate comprehensive analysis report
   - Provide overall dramatic structure evaluation

Output Progress:
[25%] Text preprocessing complete...
[50%] Parallel analysis in progress...
[75%] Result integration...
[100%] Analysis complete!
```
